Пристрій зберігання інформації - мікроелектронний пристрій, що призначений для зберігання інформації та керування процесами обміну цією інформацією у складі комп’ютерної системи.
Швидкість передачі даних - кількість даних, які можуть бути передані пристроєм зберігання інформації за одиницю часу.
Інтерфейс - спосіб з'єднання та обміну даними між пристроєм зберігання інформації та іншими компонентами комп'ютера.
Формфактор - фізичні розміри та форма пристрою зберігання інформації, які визначають його сумісність з різними комп'ютерними системами та корпусами.
Ємність - максимальна кількість даних, яку може зберігати пристрій зберігання інформації.
NAND Flash - тип флеш-пам'яті, що широко використовується в твердотілих накопичувачах (SSD) та інших пристроях зберігання інформації для зберігання великих обсягів даних.
Розділ - логічна частина пристрою зберігання інформації, що використовується для організації даних.
SSD (Solid State Drive) - твердотілий накопичувач, пристрій зберігання даних, що використовує флеш-пам'ять для зберігання даних без рухомих частин, що забезпечує високу швидкість передачі даних та надійність.
HDD (Hard Disk Drive) - жорсткий диск, традиційний пристрій зберігання даних, що використовує магнітні диски для зберігання даних з рухомими частинами.
USB (Universal Serial Bus) - універсальний послідовний інтерфейс для підключення периферійних пристроїв до комп'ютера, включаючи пристрої зберігання даних.
SATA (Serial ATA) - послідовний інтерфейс для підключення пристроїв зберігання даних, таких як HDD та SSD, до комп'ютера, що забезпечує високу швидкість передачі даних.
NVMe (Non-Volatile Memory Express) - протокол високошвидкісного доступу до твердотілих накопичувачів через шину PCI Express, що забезпечує низькі затримки та високу продуктивність.
DRAM (Dynamic Random Access Memory) - динамічна оперативна пам'ять, тип енергозалежної пам'яті, що використовується для зберігання даних та програм, які активно використовуються комп'ютером.
